# 瑞思连点-Ris_Click
## 简洁的鼠标连点器
由PYLXU(腾瑞思智)开发，在Github使用GPLv3协议开源
[请点击这里下载最新版本的构建](https://github.com/PYLXU/Ris_Click/releases)

## 基本说明
通过设定的频率与次数对鼠标进行模拟点击，同时支持快捷键进行开始与结束
